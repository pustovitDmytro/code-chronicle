## math

**direct import**:

```javascript
    import * as helpers from 'math'
```

### factorial

function:
calculate facorial of the number

**Parameters**

    - `num` **number** 

**Returns**

Returns **number** facorial of the number

**Examples**
*(factorial)*

```javascript
import { factorial } from 'math';

factorial(5); // 120

factorial(0); // 1


```

### fibonacci

function:
calculate Fibonacci Sequence

**Parameters**

    - `num` **number** 

**Returns**

Returns **number** Fibonacci number

**Examples**
fibonacci *(positive)*

```javascript
import { fibonacci } from 'math';

() => {
    return fibonacci(7);
}; // 13


```
